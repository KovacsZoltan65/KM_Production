# Lessons Learned

## Purpose

This permanent memory file records reusable lessons from completed work.

## How to Record Entries

Record lessons that will help future contributors make better decisions. Keep entries factual, specific, and linked to the documentation layer they affect.

## Template

```md
## YYYY-MM-DD - Short title

Context:

Lesson:

Impact:

Future AI Guidance:

Related:
```

## Examples

```md
## 2026-06-28 - Navigation prevents documentation drift

Context: AI-native documentation consolidation.

Lesson: A central `.kiro/index.md` makes it easier to keep steering, decisions, knowledge, playbooks, prompts, templates, checklists, workflows, and memory connected.

Impact: Future agents can find the right layer instead of duplicating guidance.

Future AI Guidance: Link between layers when adding or moving documentation.

Related: `.kiro/index.md`, `.kiro/checklists/documentation.md`
```

## Future AI Guidance

Use this file for lessons that improve future work but are not strict enough to be steering rules and not architectural enough to be ADRs.

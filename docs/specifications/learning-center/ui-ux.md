# UI és UX

## Cél

Ez a dokumentum a Learning Center tervezett felületeit és azokat az UX szabályokat írja le, amelyek a munkát támogató, de azt nem akadályozó segítségnyújtást biztosítják.

## Tervezett tartalom

- Learning Center Dashboard.
- Oldalspecifikus súgó drawer.
- Asszisztenciaszint-választó.
- Felhasználói profil és oldalankénti beállítások.
- Progress nézet.
- Keresési élmény.
- Ajánlások.
- Hibamagyarázó panelek.
- Admin tartalomkezelő felületek.

## Vázlatos UX szabályok

A Learning Center segítség legyen látható, de ne legyen tolakodó. Kezdő módban megjelenhet inline útmutatás és lépésről lépésre vezetés. Haladó módban főként kontextuális jelzések jelenjenek meg. Profi módban a segítség csak kérésre jelenjen meg, kivéve kritikus figyelmeztetéseknél.

Minden támogatott oldal mutassa az aktuális asszisztenciaszintet. A felhasználó az adott oldal szintjét a workflow elhagyása nélkül módosíthassa.

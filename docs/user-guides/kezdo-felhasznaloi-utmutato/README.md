# Kezdő felhasználói útmutató

Ez az útmutató teljesen kezdő felhasználóknak készült.

Kiindulási helyzet:

- frissen telepített KM_Production rendszer
- üres adatbázis
- nincs gyár, raktár, cikk, dolgozó, beszállító vagy vevő
- a felhasználó most látja először az alkalmazást

A példában a **KM Gépgyártó Kft.** egyetlen terméket gyárt: **Acél konzol**.

## Mintaadatok

| Adat               | Érték                      |
| ------------------ | -------------------------- |
| Gyár               | KM Gépgyártó Kft.          |
| Gyártási hely      | Budapest                   |
| Alapanyag raktár   | Alapanyag raktár           |
| Késztermék raktár  | Készáru raktár             |
| Tárolóhely         | A-01                       |
| Dolgozó            | Kiss János                 |
| Beszállító         | Acél Plusz Kft.            |
| Vevő               | Minta Gépipari Kft.        |
| Termék             | Acél konzol                |
| Termék cikkszám    | PRD-1001                   |
| Alapanyag 1        | 4 mm acéllemez, MAT-0001   |
| Alapanyag 2        | M8x20 csavar, MAT-0100     |
| Alapanyag 3        | Szürke porfesték, MAT-0200 |
| Gyártási mennyiség | 10 db                      |
